# mail-pachi  
合否メールを解析して、パチンコの演出として選考結果を表示するアプリです。  
Imapが使えるメールサービスのみ対応しています。  

Example
```
dotnet run --host <IMAP4_SERVER_ADDRESS> --email <EMAIL_ADDRESS> --password <PASSWORD>
```
